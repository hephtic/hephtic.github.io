Projects

Here are some of my personal projects:

1. Polymarket Data Analysis & Scraper Optimization Package

Topics: Prediction markets, on-chain data extraction, trading behavior

Methods: API scraping, data cleaning, performance optimization, modular package design, documentation

Tools: Python (requests, pandas, tqdm), SQLite, Polymarket API

Data: Polymarket event, comment, and trade data

Built modular scraping tools for collecting Polymarket prediction market data (trades, comments, prices). Streamlined and merged multiple packages into a single optimized interface for scalable analysis of trader behavior and market dynamics across thousands of markets. Used outputs to explore liquidity shifts and sentiment in U.S. election markets.

2. Income Factors Analysis

Topics: Gender wage gap, labor economics, income inequality

Methods: Linear regression, Blinder-Oaxaca decomposition, Lasso Regression, Random Forest

Tools: Python (pandas, scikit-learn, statsmodels), R (dplyr)

Data: IPUMS labor survey (2009–2024)

Analyzed gender-based wage disparities by separating explained and unexplained income gaps using statistical and machine learning models. Highlighted key predictors of wage differences and assessed their structural contributions over time.

3. Unemployment COVID Recovery Analysis

Topics: Labor market recovery, education inequality, regional disparities, COVID-19 economic impact

Methods: Linear regression (OLS), quantile regression (τ = 0.1–0.9), log transformations, bootstrapping, Breusch-Pagan test

Tools: Python (pandas, statsmodels, matplotlib, seaborn)

Data: USDA Economic Research Service – County-level education and unemployment rates (2019–2023)

Used quantile regressions to assess how education levels influenced post-COVID unemployment recovery across U.S. counties. Found stronger recovery patterns among higher-educated regions, particularly in lower quantiles of the unemployment distribution.

4. Sticky-ness of Inflation Analysis

Topics: Inflation stickiness, housing market dynamics, CPI decomposition, macroeconomic indicators

Methods: Time series decomposition, seasonality testing, cross-basket regression analysis, correlation analysis

Tools: Python (pandas, statsmodels, matplotlib)

Data: Bureau of Labor Statistics CPI data, Zillow Housing Value Index (ZHVI)

Decomposed CPI time series to measure inflation stickiness across categories. Built a custom inflation tracker for the housing sector using Zillow data, showing housing as a persistent driver of sticky inflation.

5. Evolutionary Game Theory Simulation

Topics: Evolutionary game theory, cooperation, public goods games, punishment, strategy dynamics

Methods: Agent-based simulation, heatmap analysis, variance analysis, parameter sweeps

Tools: Python (numpy, matplotlib, seaborn)

Data: Synthetic data generated from simulation runs across parameter grids

Developed an agent-based model of a finite population public goods game with optional punishment and mutation. Explored how cooperation evolves under varying selection strengths and institutional rules, identifying regimes that foster stable cooperation.
