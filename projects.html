1. Polymarket Scraper Package & Profit and Loss Engine

Topics: Prediction markets, on-chain data extraction, trader performance analysis

Methods: API scraping, data cleaning, PnL calculation logic, performance optimization, modular package design

Tools: Python (requests, pandas, tqdm), SQLite, Polymarket API

Data: Polymarket event, comment, and trade data

Built modular scraping tools for collecting Polymarket prediction market data. Streamlined multiple APIs into a single optimized Python package and engineered a high-performance (PnL) engine. This component processes hundreds of millions of trades to accurately track individual trader profitability, distinguishing between realized and unrealized PnL and handling complex settlement logic for thousands of markets.

2. From Discrete Steps to Continuous Paths: Bridging GARCH and Stochastic Volatility

Topics: GARCH, Stochastic Volatility, Diffusion Limit, Financial Econometrics, Volatility Forecasting, SDEs

Methods: Maximum Likelihood Estimation (MLE), Quasi-MLE, Rolling-Window Forecasting, Monte Carlo Simulation, Time Series Analysis

Tools: Python (Pandas, NumPy, Statsmodels, SciPy, Matplotlib), LaTeX

Data: S&P 500 Daily Returns (2000-2024)

This project provides a comprehensive bridge between the two dominant paradigms of financial time series analysis. I explored the theoretical connection where discrete GARCH models converge to continuous-time stochastic differential equations (SDEs). To ground this theory, I fitted both GARCH(1,1) and Cox-Ingersoll-Ross (CIR) models to S&P 500 data. The empirical results demonstrated that while GARCH models offer superior robustness for forecasting, SDEs provide the essential theoretical language for asset pricing. The analysis concludes that these frameworks are complementary, not competing, tools for modern finance.

3. Income Factors Analysis

Topics: Gender wage gap, labor economics, income inequality, labor market segmentation

Methods: Linear regression, Blinder-Oaxaca decomposition, Lasso Regression, Random Forest, PCA, NMF, XGBoost, Hierarchical Bayesian Linear Model (MCMC), Cramér’s V, Kruskal-Wallis H-test

Tools: Python (pandas, scikit-learn, statsmodels, scipy, xgboost), R (dplyr)

Data: IPUMS labor survey (2009–2024), CPS microdata

Analyzed gender-based wage disparities by separating explained and unexplained income gaps using statistical and machine learning models. Highlighted key predictors of wage differences and assessed their structural contributions over time.

Additionally, used unsupervised methods (PCA, NMF) to explore lower-dimensional structure in wage distributions across industries. Applied XGBoost and hierarchical Bayesian regression (via MCMC) to model wage variation and identify robust nonlinear predictors, incorporating race, education, industry, and time. Cramér’s V and nonparametric tests further revealed clustering and dispersion trends in labor market participation.

4. Unemployment COVID Recovery Analysis

Topics: Labor market recovery, education inequality, regional disparities, COVID-19 economic impact

Methods: Linear regression (OLS), quantile regression (τ = 0.1-0.9), log transformations, bootstrapping, Breusch-Pagan test

Tools: Python (pandas, statsmodels, matplotlib, seaborn)

Data: USDA Economic Research Service – County-level education and unemployment rates (2019–2023)

Used quantile regressions to assess how education levels influenced post-COVID unemployment recovery across U.S. counties. Found stronger recovery patterns among higher-educated regions, particularly in lower quantiles of the unemployment distribution.

5. Sticky-ness of Inflation Analysis

Topics: Inflation stickiness, housing market dynamics, CPI decomposition, macroeconomic indicators

Methods: Time series decomposition, seasonality testing, cross-basket regression analysis, correlation analysis

Tools: Python (pandas, statsmodels, matplotlib)

Data: Bureau of Labor Statistics CPI data, Zillow Housing Value Index (ZHVI)

Decomposed CPI time series to measure inflation stickiness across categories. Built a custom inflation tracker for the housing sector using Zillow data, showing housing as a persistent driver of sticky inflation.

6. Evolutionary Game Theory Simulation

Topics: Evolutionary game theory, cooperation, public goods games, punishment, strategy dynamics

Methods: Agent-based simulation, heatmap analysis, variance analysis, parameter sweeps

Tools: Python (numpy, matplotlib, seaborn)

Data: Synthetic data generated from simulation runs across parameter grids

Developed an agent-based model of a finite population public goods game with optional punishment and mutation. Explored how cooperation evolves under varying selection strengths and institutional rules, identifying regimes that foster stable cooperation.
