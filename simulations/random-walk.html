Random Walk Recurrence

In a symmetric random walk, you take equal-sized steps in random directions. The surprising twist is how dimension changes the story: in 1D and 2D, you will almost certainly return to your starting point if you walk forever (recurrence). In 3D and higher, there’s a real chance you’ll never come back as the path can drift away indefinitely (transience).

This simulation only uses a finite number of steps, so recurrence isn’t guaranteed, but you’ll still see how return probabilities behave differently across dimensions.


    Notice how in 1D and 2D the return rates approach 1 (this is not fully convincing in the 2D case if we had infinite computing power it would be true), while the return rate in 3D seems to converge at around 0.34. This is the convergence of probabilities as we approach infinity supported by the recurrence and transience in these dimensions.
