Random Walk Recurrence

In probability theory, a symmetric random walk is recurrent in 1D and 2D, as in it returns to the origin almost surely, given infinite time. In 3D (and in any higher dimensions), this random walk is transient, there's a nonzero chance it never returns. This simulation uses finite steps for computation purposes, so recurrence is not guaranteed, but you'll see strong differences in return behavior and convergence of probability as we simulate random walks in each of these dimensions.


    Notice how in 1D and 2D the return rates approach 1 (this is not fully convincing in the 2D case if we had infinite computing power it would be true), while the return rate in 3D seems to converge at around 0.34. This is the convergence of probabilities as we approach infinity supported by the recurrence and transience in these dimensions.
